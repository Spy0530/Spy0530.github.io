CentOS7.6下载与安装配置多节点

Author: Spy Date: September 10, 2020  10:44:23 Category: 环境搭建 操作系统 CentOS

  • 版本介绍

    CentOS Stream 与 CentOS Linux 的区别

    CentOS Stream

Centos Stream是一个滚动发布的Linux发行版，它介于Fedora Linux的上游开发和RHEL的下游开发之间而存在。你可以把CentOS Streams当成是用来体验最新红帽系Linux特性的一个版本，尝鲜使用

CentOS Linux

CentOS Linux 就是普通使用的CentOS的系统了，如果追求稳定性，和正式使用，日常使用，还是强力推荐使用这个的

  • 下载

    CentOS7.6不再更新

不更新的镜像下载地址： http://vault.centos.org/ 点此访问

还在更新的镜像下载地址： http://mirrors.aliyun.com/centos/ 点此访问

  • 安装

  • 软件选择 （开发及生成工作站）

    • 附加开发
    • DNS名称服务器
    • 开发工具
    • Emacs
    • FTP服务器

    • 主框架访问
    • MariaDB数据库服务器
    • 网页用Perl
    • 平台开发
    • Python

  • 安装目标位置

    • 自动配置分区（默认）

  • 配置网络

  1. 改虚拟机网络属性为桥接
  2. 打开CentOS终端 切换为管理员账号 su root
  3. 使用工具分配IP dhclient
  4. 查看分配的固定IP ifconfig (节点1为例： 192.168.1.158)
  5. 将分配的IP写入网络配置文件 vim /etc/sysconfig/network-scripts/ifcfg-ens33
  6. 修改 BOOTPROTO=dhcp 为 BOOTPROTO=static 修改 ONBOOT=no 为 ONBOOT=yes 增加IPADDR=192.168.1.158,NETMASK=255.255.255.0,GATEWAY=192.168.1.1,DNS1=119.29.29.29
  7. 保存后重启配置文件 systemctl restart network.service

  • 远程连接

    在Windows 端使用xshell进行连接时如果出现汉字乱码可修改： 文件–属性–终端 找到编码改为UTF-8即可。
